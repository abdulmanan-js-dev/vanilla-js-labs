## 📌 Featured Projects

### 🧮 Modular JavaScript Calculator

A fully modular calculator built without using `eval()`, focusing on clean architecture and expression evaluation.

**Key Features:**

-   Supports +, −, ×, ÷, % operations
-   Smart input validation
-   Multi-file modular structure (ES Modules)
-   Custom expression evaluator

-   Clean and interactive interface
